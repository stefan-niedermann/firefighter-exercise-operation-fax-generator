Automatische Koordinatensuche

Führt automatisch eine Suche nach X- und Y-Koordinaten durch und verwendet diese im Übungs-Fax.
Zu diesem Zweck wird der angegebene Ort an den Server von OpenStreetMap übertragen.
Durch Aktivierung stimmen Sie der Datenschutzerklärung von OpenStreetMap zu, die Einstellung wird in diesem Browser gespeichert.

Ist die automatische Koordinatensuche deaktiviert oder werden für den Ort keine Koordinaten gefunden,
wird als Standardwert ein Punkt in Deutschland verwendet.
